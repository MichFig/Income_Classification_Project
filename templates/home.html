Income Classification Project

By: Connor MacKenzie, Ismahan Ahan, Michelle Fegatelli

Project Focus

Examination of 1994 Census Income data in an attempt to predict whether an individual's income will exceed $50,000.00 per annum based on a variety of factors such as education, age, marital status, relationship status and work classification.

The goal of the project is to the Scikit-Learn and Tensorflow libraries in Python to create a variety of machine learning models, and find the most accurate model to deploy on this site where users can make their own predictions from the options of the dataset.

Preparing the Data

Many of the column names and individual data entries had leading whitespace, so this was stripped by renaming the columns, and looping through the entries.

To create charts to see any potential relationships between certain columns and the income, the pandas groupby function was used and columns generated for a total count of the rows, number earning <=50K, and number earning >50K.
